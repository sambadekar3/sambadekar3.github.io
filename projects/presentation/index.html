Design Strategy: I started with prototyping the interactions instead of mapping visuals steps.
Project: Tab + Type
Understanding The Problem:
In the older experience, the only way to edit text of a tag was through the right panel.
Project: Tab + Type
Design Challenges: Drag vs Edit mode, Overriding 'delete' key.

Interactive Area: Evolving Affordances
