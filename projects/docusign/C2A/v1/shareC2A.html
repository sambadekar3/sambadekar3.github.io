Implementation

Modals are created by calling the javascript function Olive.modal.open(options), where options is a configuration object (full details below). A unique identifier must be provided as options.name, and this will be the name of your instance. Provide the content of your modal with options.content, which can be a string of HTML, a DOM object, a jQuery object, or a compiled Angular template. If you have not included the .modal wrapper it will be created automatically. The close X and the modal's message section will also be added automatically.

Modals can close one of two ways, with instance.complete() or instance.escape(). These are used to represent whether the user completed the process successfully or if the user decided to abort the process. Each modal includes two arrays of functions, the onComplete queue and onEscape queue. When the modal completes or escapes, the appropriate queue of functions will run. Helpers to add new functions to the instance's queues are described in the code reference at the bottom of the page.
