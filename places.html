Places: Web Application for Location Based Search


Goal

To reduce the complexity of searching local businesses.

About

A week long project that re-imagines how location based search work.

Existing Applications

There are quite a few applications that allow us to search for nearby businesses. But in the process of adding more features the app itself gets bloated with a bunch of settings and filters.

Design

Ideation and initial mockups

After going through few design iterations we finalized upon few graphs for evaluation.

Application

The visualization system is implemented using D3.js. The data used is from public repositories of company 'Balanced' on GitHub.
